# adf_calculadora_imc

Projeto de estudo da Academia do Flutter.


# Como rodar: 
1. Git clone do projeto;
2. No diretorio do projeto execute o comando `flutter create .` para criar demais estruturas
3. `flutter pub get`
4. `flutter run`
